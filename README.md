## 🏗️ Deployment

To host this portfolio yourself:

**1. Clone the repository:**

```bash
git clone https://github.com/PaterneMpo/My-Cv.git
```

**2. Open with a Live Server:**

Open `index.html` in your browser. No build step required — Tailwind is served via CDN for maximum portability.

**3. Deploy to GitHub Pages:**

Push to your `main` branch and enable GitHub Pages in the repository settings.
